# Specification: Create the Spanish AI Agent Learning Guide

## Overview
Consolidate and improve the AI Agent guide content into a single, comprehensive markdown file named `My-AI-Learning-Guide-ES.md` written in Spanish. This file will serve as the primary educational resource and reference for beginners, advanced engineers, and full-stack developers.

## Requirements
- Create `My-AI-Learning-Guide-ES.md` based on `AI-Agent Guide.md` and `spec/initial.md`.
- Content MUST be entirely in Spanish, maintaining an educational and conversational tone.
- Code snippets and technical identifiers MUST remain in English.
- Use Obsidian-flavor callouts (`> [!NOTE]`, `> [!WARNING]`, etc.).
- Ensure there are consistent sections for each tool (Cursor, Antigravity, Gemini CLI, OpenCode, Claude Code, Codex CLI).
- Include comprehensive sections on Skills (with examples) and MCP (configurations, security, exclusions).
- Add model evaluation data for coding, documentation, generation, etc.

## Key Sections
1. **Introducción y Conceptos Básicos:** Memoria persistente, niveles de contexto (AGENTS.md, GEMINI.md).
2. **Skills:** Profundidad técnica, mejores prácticas, ejemplos de estructura.
3. **MCP (Model Context Protocol):** Aspectos de seguridad, configuración y herramientas.
4. **Plugins y Extensiones:** Resumen de integración para las distintas CLIs.
5. **Hooks:** Configuraciones y diferencias (Gemini CLI vs Claude Code).
6. **Subagentes:** Consejos y arquitectura.
7. **Automatización:** Headless mode y scripting.
8. **Evaluación de Modelos:** Comparación de Gemini, Claude, y OpenAI GPT-5/4o.

## Acceptance Criteria
- File is properly formatted in Markdown.
- Language consistency rules are strictly followed.
- All requested tools and concepts are documented according to `product-guidelines.md`.
